## Padroes principais

- JavaScript simples e direto.
- CSS escopado por tela/modulo.
- Reutilizar componentes existentes.
- Evitar duplicidade de funcoes.
- Evitar CSS em camadas conflitantes.
- Usar formatadores centralizados para moeda, numero e texto.
- Nao misturar ajuste visual com regra de negocio.
- Supabase e a fonte principal de dados.
